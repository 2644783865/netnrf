## 项目结构
- Netnr.Core 类库
- Netnr.Data 数据访问、仓储
- Netnr.Domain 实体
- Netnr.Func 应用
- Netnr.ResponseFramework Web站点

----------
## 数据表
- 系统用户（SysUser）
- 系统角色、角色权限（SysRole）
- 系统菜单（SysMenu）
- 系统按钮（SysButton）
- 系统日志（SysLog）
- 表配置（SysTableConfig）
----------

## 功能
- 表格：动态配置标题、宽度、排序、对齐方式、格式化、冻结、点击排序等
- 表单：动态生成表单，自定义标题、排序、跨列、类型、必填等
- 查询：动态生成查询面板，自定义哪些字段支持查询
- 等等
